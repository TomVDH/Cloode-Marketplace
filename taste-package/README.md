# Taste Claude

Premium frontend design skills that override default AI biases. Enforces high-end typography, calibrated color, asymmetric layouts, motion choreography, and anti-generic UI standards.

## Skills

| Skill | What it does |
|-------|-------------|
| **design-taste-frontend** | Senior UI/UX engineer skill with tunable dials (variance, motion, density). Enforces strict component architecture, CSS hardware acceleration, anti-slop patterns, and Bento motion paradigms. The core skill for building new interfaces. |
| **high-end-visual-design** | $150k agency-level design directive. Creative variance engine with three vibe archetypes (Ethereal Glass, Editorial Luxury, Soft Structuralism) and three layout archetypes. Double-bezel component architecture and cinematic motion choreography. |
| **minimalist-ui** | Clean editorial-style interfaces. Warm monochrome palette, typographic contrast with serif/sans pairing, flat bento grids, muted pastels. No gradients, no heavy shadows. Notion-meets-Linear aesthetic. |
| **industrial-brutalist-ui** | Raw mechanical interfaces fusing Swiss typographic print with military terminal aesthetics. Rigid grids, extreme type scale contrast, utilitarian color, analog degradation effects (halftones, CRT scanlines, dithering). |
| **redesign-existing-projects** | Upgrades existing websites and apps to premium quality. Audits current design against a comprehensive checklist, identifies generic AI patterns, and applies targeted fixes without breaking functionality. |
| **stitch-design-taste** | Generates DESIGN.md files optimized for Google Stitch screen generation. Translates the anti-slop directives into Stitch's semantic design language. |
| **full-output-enforcement** | Overrides default LLM truncation behavior. Bans placeholder patterns, enforces complete code generation, and handles token-limit splits cleanly. |
| **find-skills** | Helps discover and install agent skills from the open skills ecosystem via the Skills CLI (`npx skills`). |

## Usage

Each skill triggers automatically based on context. The design skills layer together — `design-taste-frontend` provides the engineering foundation, while the aesthetic skills (`high-end-visual-design`, `minimalist-ui`, `industrial-brutalist-ui`) provide specific visual directions.

Use `redesign-existing-projects` when upgrading an existing codebase rather than building from scratch.

Use `full-output-enforcement` alongside any design skill to prevent truncated or placeholder-filled output.
